## Overview

TaskFlow is a modern productivity web application designed to provide an intuitive and elegant task management experience. It combines task organization, analytics, and calendar-based planning within a premium glassmorphism interface.

Built entirely using vanilla web technologies, the application delivers smooth interactions, responsive layouts, and persistent local storage without requiring any backend or external database.

---

## Features

### Task Management

- Create, edit, and delete tasks
- Mark tasks as completed
- Real-time search functionality
- Filter tasks by:
  - All
  - Active
  - Completed
- Difficulty levels:
  - Hard 🔴
  - Medium 🟡
  - Easy 🟢
- Optional deadline support
- Overdue task detection
- Persistent storage using Local Storage

### Analytics Dashboard

- Interactive Doughnut Chart powered by Chart.js
- Difficulty distribution visualization
- Completion percentage tracking
- Animated progress bar
- Statistics cards for:
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
  - Overdue Tasks

### Calendar Integration

- Interactive monthly calendar powered by FullCalendar
- Month navigation controls
- Today's date highlighting
- Selected date highlighting
- Glowing deadline indicators
- Date-based task inspection:
  - Tasks created on selected date
  - Tasks with deadlines on selected date

### Smart Notifications

- Browser notification support
- Deadline alerts
- In-app toast notifications
- Dynamic notification badge

### Premium User Interface

- Glassmorphism design
- Dark mode and light mode
- Smooth animations
- Responsive layout
- Mobile and desktop support
- Modern typography using Inter
- Minimal and distraction-free experience

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd TaskFlow
```

Launch the application by opening:

```text
intro.html
```

or

```text
index.html
```

in any modern browser.

No additional setup, package installation, or build tools are required.

## Data Persistence

TaskFlow stores all information locally using the browser's Local Storage.

This means:

- No server is required
- No account creation is needed
- Data remains available after refreshing or reopening the browser
- Data is private to the current browser and device
